->  версия 1.1.0
1. в данной версии была добавлена возможность вствалять фотографии через ctrl + v 
2. была добавлена система backup, которая отображает последнее изображение, которое было изображено, сохраняет данные в папку C:/PictureLook
3. Был фукнционал для кнопки "приоритет", теперь кнопка имеет 2 режима включена и выключена, где при первом нажатии она переключается в режим приоритет, и загорается красным цветом в данном режиме, приложение висит надо всеми другими приложениями, при повторном нажатии, приложение будет вести себя как обычное приложение.
4. был добавлен backup для флажкового поля, "подсиськовать"

->  версия 1.0
Данная версия умеет показывать картинки которые могут быть импортированы через перетаскивание и через кнопку показать. 
Есть режим приоритет, который позволяет приложению находиться над другими приложениями, кнопка имеет зелёный цвет, и работает только один раз.
есть кнопка фон, которая меняет задний фон, каринкосмотра "будет работать если картинка имеет свойства прозрачности"
Есть галочка "подсиськовать", которая отвечает за растягивание картинки во весь интерфейс приолжения.
Есть ярлычёк вверху и название, (имеет пока раобчую версию)
